API сокращения ссылок

Сервис sua.su предоставляет открытый доступ к своему API, с помощью которого можно создавать сокращенные ссылки удаленно.

Доступ к API серверу осуществляется по протоколу http(-s) на домене api.sua.su. На каждый запрос отправляется JSON. Среди полей в ответе сервера есть "code", в котором передается код ответа сервера, и поле "status", где на человекопонятном языке описывается код сервера.

Перед использованием сервиса необходимо получить ключ доступа (token) с помощью запроса к методу user.token

Пример запроса на получение токена: https://api.sua.su/user.token

В ответ на такой запрос сервер вернет JSON с двумя полями: token - сам ключ доступа и deleteTime - время действия ключа в секундах (обычно 7 дней)

После получения ключа доступа можно взаимодействовать с остальными методами API

Для сокращения ссылки используйте метод url.set. Этот метод принимает два обязательных параметра (token - ключ доступа, url - сокращаемая ссылка) и четыре необязательных (secure - защитить ссылку паролем или нет (либо 0, либо 1, если 1, то в аргументе pass передается пароль), stat - собирать ли статистику переходов (0 или 1), time - время действия ссылки в секундах, по умолчанию 50 лет)

Коды ответов сервера:

0 : "OK" : Операция прошла успешно.

11 : "No "url" passed" : Не передана сокращаемая ссылка

12 : "No "token" passed" : Не передан ключ доступа

13 : "No "url" and "token" passed" : Ни ключб ни ссылка не переданы

14 : "Invalid URL" : некорректная ссылка

15 : "Invalid Token" : некорректный ключ

16 : "Forbidden" : доступ запрещен

17 : "Token is stale" : Срок действия ключа доступа истек

18 : "No "pass" passed" : Не передан пароль

19 : "Password too short" : Пароль короче 4 символов

20 : "Invalid time" : Некорректный формат времени
